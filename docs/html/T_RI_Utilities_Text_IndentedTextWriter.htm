Remarks

Auto-indentation means that each line written to the encapsulated TextWriter starts with a defined indentation.

 Important
IndentedTextWriter only works reliable with new line strings of LF or CRLF (see NewLine).
Examples
C#
Copy
 1// create a string builder to write into
 2var sb = new StringBuilder();
 3using(var sw = new StringWriter(sb))
 4{
 5       // create the indented text writer
 6       using(var itw = new IndentedTextWriter(sb))
 7       {
 8           // lets use tabs for indentation
 9           itw.IndentString = "\t";
10
11           // write some lines
12           itw.WriteLine("Line 1");
13           itw.IndentLevel++;
14           itw.WriteLine("Line 2");
15           itw.IndentLevel++;
16           itw.WriteLine("Line 3");
17           itw.IndentLevel--;
18           itw.WriteLine("Line 4");
19           itw.IndentLevel = 0;
20           itw.WriteLine("Line 5");
21       }
22}
23
24// get final string
25var text = sb.ToString();
26
27// result:
28// Line 1
29//     Line 2
30//         Line 3
31//     Line4
32// Line 5
