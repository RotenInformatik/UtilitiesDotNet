UncloseableStream Class

Remarks

UncloseableStream can be helpful in situations where you pass a stream to an object or method which, after it has done its job, closes the stream and you need the stream still open afterwards.

Thread Safety
Static members of this type are not safe for multi-threaded operations. Instance members of this type are not safe for multi-threaded operations.
