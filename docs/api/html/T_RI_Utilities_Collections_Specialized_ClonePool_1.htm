Remarks

The prototype can be any object which implements ICloneable or ICloneableT. Clone or Clone is used to create new items from the prototype.

The prototype itself is never used as an item or taken from the pool respectively, it is only used for cloning.

This pool implementation supports IPoolAware.

See PoolBaseT for more details.
