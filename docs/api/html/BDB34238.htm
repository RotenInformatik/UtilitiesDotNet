The function used to calculate the hash value of an object.
Remarks

This delegate always points to the function which eventually does the calculation of the hash value. If the EqualityComparisonT was constructed specifying a hash function (using the EqualityComparisonT(FuncT, T, Boolean, FuncT, Int32) constructor) the delegate points to that function, otherwise it points to an anonymous function which encapsulates T.GetHashCode.
