IniSectionAddMode Enumeration

Defines how sections are added to an IniDocument using AddSection(String, IniSectionAddMode, IDictionaryString, String) and AddSection(String, IniSectionAddMode, IDictionaryString, IListString).

public enum IniSectionAddMode
Members
  Member nameValueDescription
AppendEnd0 Appends the section, including a new section header, at the end of the IniDocument.
AppendSame1 Appends the section, including a new section header, after the last section of the same name.
MergeSame2 Adds the elements of the section at the end of the first already existing section with the same name.
Remarks

AppendSame: If no other section with the same name exists, the behaviour is the same as AppendEnd.

MergeSame: If no other section with the same name exists, the behaviour is the same as AppendEnd.
