Usage Note

In Visual Basic and C#, you can call this method as an instance method on any object of type Random. When you use instance method syntax to call this method, omit the first parameter. For more information, see Extension Methods (Visual Basic) or Extension Methods (C# Programming Guide).
Exceptions
ExceptionCondition
ArgumentNullExceptionrandomizer or list is null.
ArgumentOutOfRangeExceptionindex or count is less than zero or specifies a range outside the size of the list.
Remarks

Shuffling of items is done in-place, so during shuffling some items might appear twice in the list.

This is a O(n) operation where n is count.
