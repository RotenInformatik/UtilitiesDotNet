Remarks

The list is partially shuffled. That means that each item is moved randomly away from its original index. The more shakes, the farther away an item might move from its original position. A shakes of zero means that the list remains the same and no items are shuffled.

Since the behaviour of the actual type of list is not known, the shuffling is not done in-place. Instead, all items are copied and then shuffled, the list cleared, and then all sorted items are added again.
