Remarks

FilePath uses PathProperties to extract and store path information. See PathProperties for more details about the supported types of file paths.

FilePath provides more file path specific functionalities compared to String and offers a more consistent way of working with paths than Path. It can be implicitly converted to a String to work seamless with APIs using String for paths.

See PathProperties for possible format strings for ToString(String).

21var modRel   = modDir.MakeRelativeTo(otherDir);          // ..\..\data\mods
