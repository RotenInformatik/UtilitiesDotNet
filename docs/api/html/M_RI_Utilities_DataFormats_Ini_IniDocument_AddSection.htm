Remarks

If sectionName is null, the values are added outside a section (that is: before the first section header or at the end if no section header exists).

mode is ignored if sectionName is null, the behavior of AppendSame is applied.

If values is empty, an empty section is added anyways (consisting only of the section header).

All values in values will be added, even if a name-value-pair with the same name already exists.

This method is used if the dictionary can have one or multiple values per name-value-pair (using the inner list).
