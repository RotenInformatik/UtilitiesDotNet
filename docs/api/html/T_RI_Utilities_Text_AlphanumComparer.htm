Remarks

Natural alphanumeric sorting is a more sophisticated string sorting algorithm which respects the natural way humans look at strings, especially if they contain numbers.

The following list of strings is used as an example: Test100xyz, Test200abc, Test99abc. Traditional sorting would sort this list as shown, putting Test99abc at the end (because the first four characters are the same and then 9 is greater than 2). With natural alphanumeric sorting, the list is sorted as: Test99abc, Test100xyz, Test200abc (because the algorithm builds chunks and compares those instead of character-by-character).

Because this kind of sorting depends on the used culture, the used culture must be specified.
