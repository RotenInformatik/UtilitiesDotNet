Remarks

The EqualityComparisonT created with this constructor uses the custom hash value function as specified by hasher for calculating hash values. This means that the hash value generated by hasher must correspond to the equality comparison performed by comparer.

Therefore, the EqualityComparisonT created with this constructor can be used in scenarios which uses hash values, e.g. as the equality comparer of a HashSetT or DictionaryTKey, TValue.
