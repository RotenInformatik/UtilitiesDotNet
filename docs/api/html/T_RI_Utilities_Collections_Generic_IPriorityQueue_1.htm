Remarks

A priority queue stores items sorted by their assigned priority. The priority is assigned to an item when the item is added to the priority queue using Enqueue(T, Int32). The higher the priority, the earlier the item is dequeued (highest priority, first out). For items of the same priority, the order in which they are added is maintained (first in, first out).

null are valid item values if T is a reference type.
