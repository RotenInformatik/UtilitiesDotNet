InvalidPathArgumentExceptionfileNameWithoutExtension plus the existing extension do not form a valid new file name.
Remarks

If fileNameWithoutExtension is an empty string, the resulting file name will consist of only the extension (including its dot). If fileNameWithoutExtension is null, the resulting file name will consist of only the extension (without its dot).
