RI.Utilities.Text Namespace

Contains utilities and extensions for working with text data and I/O.
Classes

StringExtensions

StringSplitPredicate
Defines a delegate which can be used with the StringExtensions.SplitWhere(String, StringSplitOptions, StringSplitPredicate) and StringExtensions.SplitWhere(String, StringSplitPredicate) method to implement custom splitting of strings by testing between each character of the string whether it is a split position.
