Remarks

DirectoryPath uses PathProperties to extract and store path information. See PathProperties for more details about the supported types of directory paths.

DirectoryPath provides more directory path specific functionalities compared to String and offers a more consistent way of working with paths than Path. It can be implicitly converted to a String to work seamless with APIs using String for paths.

See PathProperties for possible format strings for ToString(String).

See FilePath for an example how to use DirectoryPath and FilePath.
