HeavyThreadStart Method

Starts the thread.

Namespace:  RI.Utilities.Threading
Assembly:  RI.Utilities (in RI.Utilities.dll) Version: 0.0.0.0
Syntax
C#
Copy
public void Start()
Exceptions
ExceptionCondition
InvalidOperationException The thread is already running.
TimeoutException The thread failed to return from OnBegin within Timeout.
HeavyThreadException An exception occurred inside the thread during execution of OnBegin.
Remarks

The start sequence goes as follows:

  1. Thread is created and initialized.

  2. OnStarting is called.

  3. Thread actually starts executing.

  4. Start is waiting until OnBegin finishes inside Thread or the timeout specified by Timeout occurs.

  5. OnStarted(Boolean) is called.

  6. Start returns while OnRun is executed inside Thread.

 Note
If OnRun returns before Stop was called, the thread sleeps and OnEnd is not executed until Stop is called.
