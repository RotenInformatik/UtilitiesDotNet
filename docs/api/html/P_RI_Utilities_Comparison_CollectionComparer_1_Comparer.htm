Remarks

This delegate always points to the function which eventually does the comparison between two elements of the collections being compared. If CollectionComparerT is constructed using an IEqualityComparerT or is using default equality comparison (using EqualityComparerT.Default), the delegate points to the Equals(T, T) method of that equality comparer. If CollectionComparerT is constructed using a comparison function, the delegate points to that function. If the CollectionComparerT uses reference equality, the delegate points to an anonymous function which encapsulates Object.ReferenceEquals(Object, Object).
