Remarks

The swapping is done by copying the first and second item, then replacing the first and second item with the value of defaultValue, and then replacing the first and second item with the swapped and copied items. Therefore, briefly during the swapping, the list does not contain the first and second item but twice the value of defaultValue in their place.
