Parameters

text
Type: SystemString
The text to write (can be null to write an empty string).
encoding
Type: System.TextEncoding
The encoding used to write the file.

Return Value

Type: Boolean
true if the file was newly created, false if it already existed and was overwritten with the specified text.
Exceptions
ExceptionCondition
InvalidOperationException The file contains wildcards.
IOException The file is in use.
UnauthorizedAccessException The user does not have the required permissions, the file is read-only, or the file is an executable which is in use.
PathTooLongException Although being a valid file path, the file path is too long for the current system to be used.
DirectoryNotFoundException The files directory does not exist or is not available.
NotSupportedException The file is not of a compatible path type used on the current system.
Remarks

The current systems default encoding (Default) is used if encoding is null.

The file is created if it does not already exist. If it already exists, the file is overwritten with a new file.
