Remarks

Besides wrapping the request to process (Request), RequestResponseItemTRequest, TResponse also provides methods to control and release the request processing, such as Respond(TResponse) or Cancel.

See RequestResponseCollectionTRequest, TResponse, TItem for more details.

Thread Safety
Static members of this type are safe for multi-threaded operations. Instance members of this type are safe for multi-threaded operations.
