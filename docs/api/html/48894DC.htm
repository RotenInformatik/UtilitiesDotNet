public Exception ThreadException { get; }

Property Value

Type: Exception
The exception of the thread or null if no exception occurred, the thread was not started, or the thread is still running.
Remarks

The value of ThreadException is reset to null by Start and set during the execution of the thread if an exception occurs.

ThreadException is set for any unhandled exception which occurs in the thread (OnBegin, OnRun, OnEnd), except for ThreadAbortExceptions.
