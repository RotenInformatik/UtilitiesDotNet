Remarks

PathProperties analyzes a path and contains, besides the path itself, several properties which describe the characteristics of the path. For invalid paths, an error description is provided (Error).

PathProperties supports Windows, Unix, and UNC style paths.

ToString(String) supports the following format strings; o, O, g, G, null or empty string: The original path (PathOriginal) / n, N: The normalized path (PathNormalized) / r, R: The resolved path (PathResolved).

Thread Safety
Static members of this type are not safe for multi-threaded operations. Instance members of this type are not safe for multi-threaded operations.
