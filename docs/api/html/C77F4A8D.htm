Remarks

This order comparer can be used if an IComparerT or IComparer is required but order comparison should be handled by a custom function or lambda expression.

Examples
C#
Copy
1// create a list of strings we want to sort
2var names = new List<string> { "Andrew", "Charles", "Bob" };
3
4// create the comparer
5// what we do: longer strings are always "greater than" shorter strings, strings of equal length use default case-insensitive comparison
6var comparer = new OrderComparison<string>((x,y) => x.Length != y.Length ? x.Length.CompareTo(y.Length) : StringComparer.OrdinalIgnoreCase.Compare(x, y));
7
8// sort the strings based on our customized comparison
9names.Sort(comparer);
