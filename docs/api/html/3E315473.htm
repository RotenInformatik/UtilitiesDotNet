Remarks

The behaviour of this method is controlled by the presence or absence of the DoNotGetHashFromElements option.

objis enumerated exactly once. obj is not enumerated if the hash code is taken from the collection itself.

This is a O(1) operation if the hash code is taken from the collection itself, O(n) otherwise where n is the number of items in obj.
