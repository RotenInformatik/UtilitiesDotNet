Remarks

BinaryStream is used when a BinaryReader or BinaryWriter needs to be used as a stream.

A BinaryStream can either support reading (using a BinaryReader) or writing (using a BinaryWriter) but not both at the same time.

Thread Safety
Static members of this type are not safe for multi-threaded operations. Instance members of this type are not safe for multi-threaded operations.
