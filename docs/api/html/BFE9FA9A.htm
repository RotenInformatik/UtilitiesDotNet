Remarks

InvariantCultureIgnoreCase is used for comparison.

All elements in all sections matching the specified section name are sorted. If section is null, the elements outside any sections are sorted.

 Important
Sorting of INI elements in a section does only work reliable if a section only contains ValueIniElements.
