The NotAnEnumerationArgumentException is thrown when an argument is not of an enumeration type.
NotFiniteArgumentException
The NotFiniteArgumentException is thrown when a floating point argument is not a finite number (means: it is "NaN"/"Not-a-Number" or infinity (either positive or negative)).
NotReadableStreamArgumentException
The NotReadableStreamArgumentException is thrown when a Stream argument cannot be read.
NotSeekableStreamArgumentException
The NotSeekableStreamArgumentException is thrown when a Stream argument cannot be accessed randomly.
NotWriteableStreamArgumentException
The NotWriteableStreamArgumentException is thrown when a Stream argument cannot be written.
