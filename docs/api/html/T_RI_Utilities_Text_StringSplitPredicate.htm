StringSplitPredicate Delegate

Defines a delegate which can be used with the StringExtensions.SplitWhere(String, StringSplitOptions, StringSplitPredicate) and StringExtensions.SplitWhere(String, StringSplitPredicate) method to implement custom splitting of strings by testing between each character of the string whether it is a split position.

Remarks

Split testing always happens between characters so that when a split position is found (the return value of StringSplitPredicate is true), the character at the index previous goes into the previous token while the character at the index next goes into the next token.

StringSplitPredicate is called before the first character, between each following character, and after the last character of a string to test whether to split the string at that position. Therefore, before the first character, previous is -1 and next is 0, after the first character previous is 0 and next is 1, and so forth, and after the last character previous is n-1 and next is n where n is the length of the string.
