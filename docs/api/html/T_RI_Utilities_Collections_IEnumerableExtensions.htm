Remarks
 Important
The complexity stated for the operations provided by this class are under the assumption that enumerating an IEnumerableT has a complexity of O(n) where n is the number of elements in the sequence.
Thread Safety
Static members of this type are not safe for multi-threaded operations. Instance members of this type are not safe for multi-threaded operations.
