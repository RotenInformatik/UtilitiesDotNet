Overview

RI.Utilities is a library of assorted utilities, extensions, and helpers for .NET.

It consists of a single assembly (RI.Utilities.dll) which is based on the .NET Standard 2.1 and has no other dependencies.

It is licensed under Apache 2.0.

Documentation is available, built with Sandcastle Help File Builder.
