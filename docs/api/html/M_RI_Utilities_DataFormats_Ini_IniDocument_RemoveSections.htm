The name of the sections to remove (can be null).

Return Value

Type: ListIniElement
The list of INI elements which were removed from Elements. An empty list is returned if no elements were removed.
Remarks

If sectionName is null, all values outside a section are removed (that is: before the first section header or all if no section header exists).
